### 25 Exercises for Learning NumPy

1. **Create a NumPy Array:** Create a 1D array of numbers from 0 to 9.
2. **Array Properties:** Print the shape, size, and data type of an array.
3. **Reshape an Array:** Convert a 1D array of size 9 into a 3x3 matrix.
4. **Indexing & Slicing:** Extract every second element from an array.
5. **Boolean Indexing:** Filter out numbers greater than 5 from an array.
6. **Element-wise Operations:** Multiply all elements of an array by 2.
7. **Matrix Multiplication:** Perform matrix multiplication on two random matrices.
8. **Identity Matrix:** Create a 4x4 identity matrix.
9. **Random Number Generation:** Generate a 5x5 array of random numbers between 0 and 1.
10. **Sorting an Array:** Sort a NumPy array in ascending order.
11. **Concatenation:** Concatenate two 1D arrays.
12. **Stacking:** Stack two 1D arrays vertically and horizontally.
13. **Statistics:** Compute the mean, median, and standard deviation of an array.
14. **Unique Values:** Find the unique values in an array.
15. **Flattening an Array:** Convert a 2D array into a 1D array.
16. **Broadcasting:** Add a scalar value to an entire array.
17. **Finding Maximum & Minimum:** Find the max and min values of an array.
18. **Transpose a Matrix:** Transpose a given 2D NumPy array.
19. **Dot Product:** Compute the dot product of two vectors.
20. **Create a Structured Array:** Define a structured NumPy array with named fields.
21. **Find Non-Zero Elements:** Identify the indices of non-zero elements in an array.
22. **Replace Elements:** Replace all odd numbers in an array with -1.
23. **Generate a Diagonal Matrix:** Create a 3x3 matrix with diagonal values of 1, 2, and 3.
24. **Cumulative Sum:** Compute the cumulative sum of an array.
25. **Histogram:** Compute a histogram of a random set of numbers.
